# 📄 Gerador de PDF para RSC do IFPR

## 📌 Sobre o Projeto

Gerar um arquivo único em PDF contendo a tabela de pontuação da RSC e todos os comprovantes devidamente organizados e com links é uma tarefa trabalhosa e, muitas vezes, se torna uma barreira para que servidores solicitem sua RSC.

Este projeto nasceu justamente dessa dificuldade, enfrentada durante o meu próprio processo de requisição da RSC. A ideia foi criar uma ferramenta simples, prática e acessível que eliminasse essa complexidade.

## 🚀 Objetivo

Facilitar a criação de um único arquivo PDF apostilado contendo:

* Tabela de pontuação da RSC
* Todos os comprovantes anexados
* Links organizados automaticamente

## 🧩 Funcionalidades

* Interface muito intuitiva
* Geração automática de PDF único e apostilado
* Suporte a anexos nos formatos:

  * PDF
  * PNG
  * JPG
* Salvamento de progresso do projeto
* Continuidade simples de edição

## 💾 Salvamento de Projetos

O sistema permite salvar o seu progresso a qualquer momento.

* Um arquivo com extensão `.rscprroj` será gerado
* Esse arquivo contém:

  * Todos os dados preenchidos
  * Todos os anexos autocontidos

Para continuar depois:

1. Abra o arquivo `.rscprroj`
2. Continue de onde parou normalmente

## 📄 Geração do PDF

Após finalizar o preenchimento e anexar todos os documentos:

1. Clique em **"Gerar PDF"**
2. O sistema irá gerar um único arquivo final apostilado

> O tamanho do arquivo gerado dependerá da quantidade e do tamanho dos anexos adicionados.
